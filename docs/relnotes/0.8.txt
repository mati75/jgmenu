jgmenu v0.8 release notes
=========================

Updates since v0.7.6
--------------------

v0.8 is a relatively stable release, containing only minor fixes.

  * lx: display {pre,ap}pend.csv files (as pmenu and xdg do)

  * pmenu: survive .desktop file without "Name" key
    
  * Add UID to socket name in order to support multi-user systems

  * pmenu, xdg: survive dead .desktop file symlink

  * pmenu: survive unicode file
